# FY24 Aspire Training: React

React notes app for FY24 aspire training program.

## Run Application
```
npm run dev
```

## Initial Setup

```
npm create vite@latest

# project name: notesy-app
# framework: react
# variant: JavaScript + SWC

cd notesy-app
npm i
npm run dev
```

## Add Dependencies

```
npm i bootstrap
npm i bootswatch
npm i react-icons
```
